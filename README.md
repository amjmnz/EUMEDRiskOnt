# EUMEDRiskOnt

**European Medical Device Risk Ontology**

A semantic model designed to represent European high-risk and *in vitro* diagnostic medical devices, along with their associated risks, incidents, and corrective actions.

## Academic Context

This repository contains the project developed for **Exercise 2 of the second continuous assessment (PEC 2)** in the *Representación del Conocimiento* (Knowledge Representation) course. It is part of the curriculum for the Bachelor's Degree in Computer Engineering at the [Universitat Oberta de Catalunya (UOC)](https://www.uoc.edu/).

## Ontology Documentation

Following the LOT methodology and FAIR principles, the complete documentation for this ontology has been automatically generated using WIDOCO. 

You can explore the classes, properties, and instances directly on our GitHub Pages:
**[View EUMEDRiskOnt Documentation](https://tu-usuario.github.io/nombre-de-tu-repo/)** 

## Tools & Technologies Used
* **Protégé:** Ontology editing and creation (OWL/XML, Turtle).
* **Pellet:** Semantic reasoning and consistency checking.
* **SWRL:** Rule-based logic implementation.
* **WIDOCO:** Automatic HTML documentation generation.
* **WebVOWL:** Ontology visualization.
* **OOPS! & FOOPS!:** Ontology evaluation and FAIR principles validation.

## Repository Structure
* `/PEC2_ontology.ttl`: The main ontology source file.
* `/myDocumentation/`: Contains the WIDOCO-generated HTML documentation.
* `/images/`: Screenshots of the WebVOWL visualization and reasoning inferences.
